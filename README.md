# Syncfusion Dashboard

A simple React admin dashboard demo built with Syncfusion EJ2 React components and Tailwind CSS. It demonstrates common admin UI patterns including charts, data grids, calendar scheduling, a kanban board, theming, and example pages (e.g., Users, Orders, Chat).

---

## Quick start

1. Install dependencies: `npm install`
2. Start the dev server: `npm start`

Open http://localhost:3000

Scripts:

- `npm start` — start development server
- `npm run build` — create production build

---

## Overview

- **Purpose:** Demo and reference implementation for a feature-rich admin dashboard.
- **Key features:** Theming (light/dark), charts (line, bar, pie, stacked), data grid, calendar/schedule, kanban board, rich text editor, and reusable layout components.
- **Technologies:** React (v17), Syncfusion EJ2 React packages, Tailwind CSS, React Router, react-icons.

---

## Project structure (brief)

- `src/components/` — reusable UI components (Header, Sidebar, Charts, etc.)
- `src/pages/` — example pages and views (Dashboard, Calendar, Kanban, Customers, etc.)
- `src/contexts/` — global context and state management
- `src/data/` — sample/dummy data for demos

---
